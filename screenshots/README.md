# Screenshots

CloudExify's submission checklist requires at least 2 screenshots in the repo:

1. `desktop.png` — full homepage at a desktop width (e.g. 1440px)
2. `mobile.png` — full homepage at a mobile width (e.g. 390px)

Add both image files to this folder before submitting. They can't be generated
automatically here since they need to capture your actual deployed site.
